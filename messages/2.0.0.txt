CodeContinue v2.0.0
===================

Improvements based on Package Control review feedback:

New Features:
• Package Settings menu now available:
  Preferences > Package Settings > CodeContinue > Settings
  Preferences > Package Settings > CodeContinue > Key Bindings

• Command Palette access to settings:
  "Preferences: CodeContinue Settings"
  "Preferences: CodeContinue Key Bindings"

• New "debug" setting (default: false)
  Enable to see detailed logging in View > Show Console
  Plugin is now silent by default

• Settings file now includes documentation comments

• 🆕New Chat feature — talk to your LLM about code:
  - Select code >> right-click >> "CodeContinue: Chat about Selection"
  - Also available via Command Palette
  - Multi-turn conversation in a dedicated buffer
  - Streaming responses with a thinking indicator
  - Conversation history maintained per session

Bug Fixes:
• Added missing "api_key" to default settings
